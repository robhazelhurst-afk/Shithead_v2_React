⚠️ Local Web Server Required!

You can't open index.html directly!
Modern browsers block ES6 modules from file:// URLs for security reasons.

✅ How to Run the Game (3 Steps):

  1. Close this browser tab
  2. Double-click the file: START_SERVER.bat
  3. A command window opens and shows: http://localhost:8000
  4. Open your browser to: http://localhost:8000
What if START_SERVER.bat doesn't work?
You need Python or Node.js installed:
  • Download Python (recommended)
  • Download Node.js

Manual Commands:

With Python:

cd C:\Claude\shithead_2
python -m http.server 8000

With Node.js:

cd C:\Claude\shithead_2
npx http-server -p 8000

Then open:

http://localhost:8000
